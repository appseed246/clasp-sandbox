# clasp-sandbox

## 前提

```
npm install -g @google/clasp
```

または npx で呼び出しできるようにする。

## sheets app

- Init

```bash
clasp create --type sheets --title clasp-sheets-sandbox --rootDir .
```

- Deploy

```bash
clasp push
```
